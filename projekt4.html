Suunatuled seljale

Sul on vaja:

  • Arduino Uno
  • Käärid
  • Juhtmeid
  • Nuppe
  • 9V patareid
  • Vest või muu soiv riideese
  • Niiti
  • Vabalt valitud LED tuled
  • Arduino IDE tarkvara


Töö käik:

  • Projekti alguses võid juba plaanida, kuhu vestile tahad panna Arduino, kuhu nupud ning tuled, et oleks juba alguses võimalik arvestada sellega, kui palju juhtmeid selle jaoks vaja läheb.

  • Kui jätsid eelmise projekti vahele ja see on sinu esimene kord Arduinoga, siis ära muretse! Kõik on lihtne. Pead vaid selle arvutiga ühendama, avama Arduino IDE, valima Tools alt õige arendusplaadi ning võid vabalt koodi kirjutama hakata.

  • Kui valid LED-i, millel on kaks ühenduskohta, siis on sellisel juhul pikem nõel positiivne. Kui valid RGY-LED-i, millel on neli nõela, siis on pikem nõel negatiivne ning teised nõelad vastavad värvidele.


  • Nupul tuleb ühendada 5V ja ground ükseise kõrvale, ning digital input tuleb panna kohe groundi vastu. Kui groundil on resistor, siis ei juhtu probleeme sellega, et nupust lahtilaskmisel ei muutu Arduino jaoks tulemus.



    • Koodi poole pealt:

    Kui avad Arduino IDE esimest korda, siis näed seal kahte funktsiooni, void setup() {} ning void loop() {}. void setup() sisse käivad kõik sellised käsud, mis seadistavad alguses erinevad pinid või seadmed. Seda koodi jooksutatakse ainult üks kord. Teise funktsiooni sisse käib kogu programmikood, mis kordub kuni Arduino väljalülitamiseni.

    Olenevalt sellest, kuhu soovid ühendada nupu ja LEDi, tuleb setupisse ka see kirjutada. Eeldame lihtsuse huvides, et nupp on kolmandas digital inputis ning LED on viiendas. Kood setupis näeb välja selline:

  • void setup() { pinMode(3, INPUT); pinMode(5, OUTPUT); }

    • Niimoodi saab Arduino aru, mida temalt nende pinide osas oodatakse. Muidugi saab konstantsed pesad ära vahetada muutujatega, kuid sel juhul peab meelde tuletama, et nende muutujate loomine peab jääma väljapoole igast funktsioonist (ehk siis võid muutujate loomise panna kohe koodi algusesse enne setupi funktsiooni.

    Input pini väärtust saab lugeda funktsiooniga digitalRead(3); ning LED-i saab põlema panna käsuga digitalWrite(5, HIGH) ning välja lülitada digitalWrite(5, LOW). Tänu C-lähedasele süntaksile saab lihtsa näitena kasutaga juba sellist koodi:

  • void loop() {
  • delay(15); // magab 15 millisekundit.
  • if (digitalRead(3) == HIGH) {
  • digitalWrite(5, HIGH);
  • } else {
  • digitalWrite(5, LOW);
  • }
  • }
